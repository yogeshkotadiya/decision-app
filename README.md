# Decision App

This web app allows browser to take decision for you.
Just a simple WebApp built on React
https://yogeshkotadiya.github.io/decision-app/

#### Build
```bash
npm build-webpack
```
#### Build Production
```bash
npm build-webpack:prod
```
#### Dev Server
```bash
npm dev-server
```
